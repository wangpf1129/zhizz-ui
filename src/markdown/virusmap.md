## 世界疫情数据可视化
此项目是我刚刚接触 Vue 时，练习的项目，以 Vue2 为基本，使用 Echarts 来将疫情情况以数据可视化的形式展示出来


### 所用到的技术

- 该项目采用Vue来搭建，
- 使用 Echatrs 来展示疫情图表，（引入方式采用 cdn引入的）
- 使用 jsonp 来获取接口
